# Marathi_TTS_System

<p> This project is an <b>Automated Marathi Text-to-Speech (TTS) System</b> 
that converts Marathi text from multiple input sources into natural, 
expressive, and regionally adaptive speech.</p>

<h2> Key Features</h2>
<p>1. Multi-Source Input </p>
<p>2. Expressive Text-to-Speech</p>
<P>3. Marathi Regional Dialect Support</P>
